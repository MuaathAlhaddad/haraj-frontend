# haraj-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

### Notes 
+ Never run `composer update` or `npm install`. Instead `composer install` and `npm ci` respectfully. 
+ Run  
  + `npm run build --mode development` to compile the assets without minified 
  + or `npm run build` to compile and minified
